# Music Mouse Web Clone

A simple webpage inspired by Laurie Spiegel's music mouse software.

![musicmouse](./public/textlogo.png)

[Developed and tested on firefox, no promises on anything else lol.]


## Controls:

- Click to toggle note triggers.
- Hover mouse over the grid to select center note (shift to disengage while hovering)
- Press \`/`~` and a number to toggle that voice.
- Press a number to select a voice:
    - `a`/`s`: make voice horizontal/vertical
    - `+`/`-`: plane voice up or down in pitch
- `q`: toggle strum
- `w`/`e`/`r`/`t`/`y`/`u`/`i`/`o`: toggle sequence note
- `z`/`x`: plane scale
- `c`/`v`/`b`: set scale major/minor/dorian
